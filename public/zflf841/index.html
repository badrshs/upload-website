Intelligent Sorting System

A sorting mechanism is a technique applied for organising matter in a systematic manner, according to certain properties. Sorting processes are commonly used in industrial manufacturing fields. Over the years, it has evolved to become an essential stage in production lines. For instance, in drug factories, they are responsible for ordering pills based on specified criterion. Another widely known example takes place categorizing waste material for recycling operations with the help of the above described procedure.

Recent technological advancements have lead to development of computerized sorting machines. These machines collect data through different types of sensors such as chemical, laser, pressure, etc. which in turn represents the conditions for distinguishing particular items.

In this project, the concept of artificial intelligence is implemented alongside optical sensors for the sake of creating reconfigurable sorting system which can be made to learn how to differentiate between requirements in the organising process.

Objective

The aim of this project is to develop an intelligent learning machine, i.e. to create an implementation of a robotic system which adopts the mechanism of supervised machine learning techniques that will be reconfigured upon placement in different environments. The methodology being applied will enable the system to sort any given items as input and adapt to the work conditions it is exposed to in order to generate the required output of arrangement, regardless of the data it is initially fed as it would have self-taught itself to cope. 

Computer Vision

Machine vision is the utilization and processing of imaging to gather information that supports the functioning of automation processes. The machine vision strategy is adopted by many inspection procedures such as counting, gauging, or detecting defects, instead of relying solely on operator judgement thus making performing the task in a more time efficient manner while also improving precision and reliability..
