  • about.html - About Artists Page View
  • about-single.html - About Us Page View
  • albums.html - Artists Albums Page View
  • album-single.html - Album Page View
  • archives.html - Main Archive Page View
  • blog.html - Blog Page View
  • events.html - Main Events Page View
  • event-single.html - Event Page View
  • gallery.html - Main Gallery Page View
  • gallery-page.html - Gallery Page View
  • videos.html - Main Videos Page View
  • contact.html - Contact Us Page View

CSS Files and Structure

To the Top

This theme has been implemented as a responsive layout. The full width of the template is fluid for screen mode, and it still has a boxed version.

The CSS file is located in the css folder. The css file is registered and included in the "head"..."/head".

CSS file is very good structured and arranged so you can search very easy:
