A) HTML Structure - top

All the pages are 980px wide. Thanks to the settings box you can change site color and background textures, the same thing for the fonts.

The code is commented so you can find everthing you need easy.

To use less space we use carousel effect on more then four posts and projects.

The site is fully responsive to four type of resolutions.
